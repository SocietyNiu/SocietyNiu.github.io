What are WeChat Mini-Programs

WeChat Mini-Programs is well known by WeChat users. If you have never used WeChat, click here to get more information.

In short, WeChat Mini-Programs are “sub-applications” within the WeChat ecosystem. You can use them like using a browser to browse the web instead of downloading an app.

WeChat Mini-Programs are written by wxml, wxss, js and json. Wxml and wxss are the specific language s based on html and css.

I have learnt WeChat Mini-Programs since sophomore year. And I have provided solutions for many enterprises and individuals. Here are some of my representative works.

My Representative Works

My representative works are made for commercial use. So it’s not suitable to post the QR codes to let you try. I will post the images of my works operating on the real phone and briefly introduce the functions to you.

An Auxiliary Tool for Start-up Team

This project was written for a senior majoring in industrial design. She gave me the UI design samples and asked me to achieve the functions by WeChat Mini-Programs. Here are the actual display effects.

The project involved following technical points and difficulties.

  • Database The core question was how to achieve data synchronization. Specifically, there are different identities like product directors or project managers. Project managers are in charge of controlling the whole process. For example, project managers need to fill in some information or stop the timer. Then others’ terminal should keep the same state. The same information should be displayed on everyone’s phone. I used the function ‘watch’ to monitor the change of database. It was hard to achieve this function. But fortunately, it worked well.
  • UI Design As you see, the UI is exquisite. Although it was not designed by me, I converted the design to code well.
  • Development Time I am proud of completing it within three weeks. And during that period, I was still in daily normal class.
  • Robustness Short development time didn’t affect the quality. The program does not have major bugs so far.

Project Management Program

This program was made for a department of a state-owned enterprise in Hangzhou. The manager needed a WeChat Mini-Program to learn about the process and other information of some projects. Here are the actual display effects. Some private information has been hidden.

The project involved following technical points and difficulties.

  • File Processing A big enterprise usually needs to deal with many files. In this program, I used node.js to handle files uploading to the servers. When you upload excel files which meet the requirements, the servers will handle them automatically, update the database and display them on everyone’s terminal.
  • Cloud Servers For the program using by a big enterprise, it is significant to guarantee the stability. I have chosen the most efficient and economical methods to organize data.

Hui Scientific Research

Hui means “can” in Chinese. This program was made for those researchers or students in the university. This is an online community to sharing knowledge. Everyone can upload videos or views to help others. Of course, you will benefit from it by gaining community virtual coins, which can be exchanged to real money or substance.

This program is not yet complete. We have a start-up team to achieve this project. I am the only programmer on the team. And others are researchers or PhD students.
